# 共有トークンの発行

## 共有トークンとは

パートナーが自分の稼働状況をディレクターに共有するための仕組みです。

- トークンはディレクターごとに個別に発行します
- 1つのトークンは1人のディレクターのみが登録できます（1回限り）
- ディレクターに見えるのは**合計稼働率のみ**で、タスク名や案件の詳細は一切見えません

## トークンの発行

![共有トークン管理画面](/images/share-tokens.png)

1. サイドバーの「**共有トークン**」をクリック
2. 「**新しいトークンを発行**」セクションで表示名を入力
   - 表示名はディレクターのワークスペースに表示される名前です（例: 田中太郎）
3. 「**発行**」ボタンをクリック

## トークンの共有

発行されたトークンをディレクターに伝えます。

1. トークン一覧の「**コピー**」ボタンをクリック
2. トークン文字列がクリップボードにコピーされます
3. メールやチャットでディレクターに送付してください

::: warning
トークンは秘密情報です。共有したいディレクター以外には伝えないでください。
:::

::: tip 同じディレクターに複数トークンを発行する
ディレクター側で複数のグループ（例: デザイナー、コーダー）にあなたを登録したい場合、グループごとにトークンを発行してください。表示名をグループに合わせて設定すると管理しやすくなります（例: 「田中太郎（デザイン）」「田中太郎（コーディング）」）。
:::

## トークンの状態

各トークンには登録状態が表示されます:

- **未登録**: まだディレクターが登録していない状態
- **登録済み（日付）**: ディレクターがワークスペースに登録した状態

## 表示名の変更

1. トークン一覧でトークンの「**編集**」をクリック
2. 新しい表示名を入力
3. 「**保存**」をクリック

## トークンの削除

1. トークン一覧の「**削除**」をクリック
2. 確認ダイアログで「OK」をクリック

::: danger
トークンを削除すると、ディレクターのワークスペースからも自動的にパートナーが消えます。同じディレクターに再度共有したい場合は、新しいトークンを発行して渡す必要があります。
:::
