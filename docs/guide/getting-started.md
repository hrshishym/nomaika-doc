# Nomaika とは

Nomaika（ノマイカ）は、フリーランスのWebコーダーやデザイナーなどの制作パートナー向けの工数管理システムです。

## こんな課題を解決します

- ディレクターから「今週空いてますか？」と毎回聞かれる
- 問い合わせのたびにスケジュールを確認して返答する手間がかかる
- 案件のアサインまでに時間がかかり、機会損失が発生する

## Nomaika でできること

### パートナーとして

1. **タスクを登録** - 案件ごとに期間と日別工数を設定
2. **非稼働日を管理** - 有給や休日を登録して正確な空き状況を反映
3. **共有トークンを発行** - ディレクターごとに個別のトークンを発行

### ディレクターとして

1. **ワークスペースにパートナーを登録** - トークンを入力するだけ
2. **稼働状況を一覧で確認** - 複数パートナーの空き状況を比較
3. **グループで整理** - 「デザイナー」「コーダー」などで分類

## 利用の流れ

```
パートナー                          ディレクター
    │                                   │
    │  1. タスクを登録                     │
    │  2. トークンを発行                   │
    │  3. トークンを伝える ──────────→    │
    │                                   │  4. アカウント登録
    │                                   │  5. トークンをワークスペースに登録
    │                                   │  6. 稼働状況を確認
```

::: tip
すべてのユーザーがパートナー機能とディレクター機能の両方を利用できます。ユーザー種別の区分はありません。
:::
