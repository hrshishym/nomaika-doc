# アカウント登録

## 新規登録

1. トップページの「**新規登録**」をクリック
2. メールアドレスとパスワードを入力
3. 「**登録**」ボタンをクリック

::: warning パスワード要件
パスワードは8文字以上で設定してください。
:::

## メール認証

登録後、入力したメールアドレスに認証メールが届きます。

1. メール内の「**メールアドレスを認証する**」ボタンをクリック
2. 認証が完了すると、タスク管理画面に遷移します

::: tip メールが届かない場合
- 迷惑メールフォルダを確認してください
- 認証メールは再送可能です。ログイン後に表示される認証画面から「**認証メールを再送信**」をクリックしてください
:::

## ログイン

1. メールアドレスとパスワードを入力
2. 「**ログイン**」ボタンをクリック

「**ログイン状態を保持する**」にチェックを入れると、ブラウザを閉じても次回アクセス時に自動でログインされます。

## パスワードリセット

パスワードを忘れた場合:

1. ログイン画面の「**パスワードを忘れた場合**」をクリック
2. 登録済みのメールアドレスを入力
3. メールに届くリンクから新しいパスワードを設定
