# アカウント登録

## 新規登録

![新規登録画面](/images/register.png)

1. トップページの「**新規登録**」をクリック
2. メールアドレスとパスワードを入力
3. 招待コードが必要な場合は、管理者から受け取った招待コードを入力
4. 「**登録**」ボタンをクリック

::: warning パスワード要件
パスワードは8文字以上で設定してください。
:::

::: info 招待コードについて
サービスの運用状況によっては、アカウント登録時に招待コードの入力が必要な場合があります。招待コードはサービス管理者から取得してください。
:::

## メール認証

登録後、入力したメールアドレスに認証メールが届きます。

1. メール内の「**メールアドレスを認証する**」ボタンをクリック
2. 認証が完了すると、タスク管理画面に遷移します

::: tip メールが届かない場合
- 迷惑メールフォルダを確認してください
- 認証メールは再送可能です。ログイン後に表示される認証画面から「**認証メールを再送信**」をクリックしてください
:::

## ログイン

![ログイン画面](/images/login.png)

1. メールアドレスとパスワードを入力
2. 「**ログイン**」ボタンをクリック

「**ログイン状態を保持する**」にチェックを入れると、ブラウザを閉じても次回アクセス時に自動でログインされます。

## パスワードリセット

パスワードを忘れた場合:

1. ログイン画面の「**パスワードを忘れた場合**」をクリック
2. 登録済みのメールアドレスを入力
3. メールに届くリンクから新しいパスワードを設定
