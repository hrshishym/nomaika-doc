# 非稼働日の管理

## 非稼働日とは

有給、午前休、私用、体調不良など、稼働できない日を登録する機能です。

- タスク管理画面のガントチャート最上部に「非稼働日」行として表示されます
- 共有ページ（ワークスペース）では、非稼働率がタスクの合計に加算されるため、休みの日も稼働が埋まって見えます

## 表示モード

サイドバーの「**非稼働日**」をクリックすると管理画面が開きます。2つの表示モードがあります:

- **ガントチャート**: タスク管理と同様の横軸＝日付の表形式
- **カレンダー**: 月カレンダー形式で非稼働日を色付き表示

画面上部のタブで切り替えできます。月の切り替えは「**←**」「**→**」ボタン、または年・月のセレクトボックスから行えます。

## 登録方法

### 期間登録（新規登録ボタン）

連続した期間の非稼働日を登録します。

1. 「**＋ 新規登録**」ボタンをクリック
2. 以下の情報を入力:

| 項目 | 説明 | 例 |
|------|------|-----|
| 名前 | 非稼働日の理由 | 有給、午前休、夏季休暇 |
| 開始日 | 非稼働の開始日 | 2026-03-10 |
| 終了日 | 非稼働の終了日（単一日なら開始日と同じ） | 2026-03-10 |
| 非稼働率 | 稼働しない割合（5%刻み、5〜100%） | 100%（終日休み）、50%（半日休み） |

3. 「**登録**」ボタンをクリック

### 繰り返し登録

「この期間の毎週水曜日を非稼働にしたい」といった設定が可能です。

1. 「**＋ 新規登録**」ボタンをクリック
2. 名前、開始日、終了日、非稼働率を入力
3. 「**特定の曜日のみ登録（繰り返し）**」にチェック
4. 登録したい曜日を選択（複数選択可）
5. 「**登録**」ボタンをクリック

![繰り返し登録](/images/nwd-create-recurring.png)

::: tip
繰り返し登録では、選択した曜日ごとに個別のエントリが作成されます。例えば「3/1〜3/31の毎週水曜」なら、3/5、3/12、3/19、3/26 の4件が作成されます。
:::

### セルクリック（日単位の登録・編集）

ガントチャート・カレンダーの各日付セルをクリックして直接登録・編集できます。

- **未登録セル**: クリック → 非稼働率を選択 → 即時登録
- **登録済みセル**: クリック → 非稼働率を変更（0%で削除）
- **期間エントリのセル**: セルクリックでは編集不可（エントリ名リンクから編集画面へ）

## 編集・削除

- ガントチャートやカレンダーのエントリ名リンクから編集画面へ遷移
- 編集画面で全フィールドの変更と削除が可能
