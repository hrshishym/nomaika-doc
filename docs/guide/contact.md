# お問い合わせ

## お問い合わせフォーム

サービスに関するご質問やご要望は、お問い合わせフォームから送信できます。

1. ページ下部やサイドバーの「**お問い合わせ**」リンクをクリック
2. 以下の情報を入力:

| 項目 | 説明 |
|------|------|
| お名前 | ご連絡先のお名前 |
| メールアドレス | 返信先のメールアドレス |
| お問い合わせ内容 | ご質問やご要望の詳細 |

3. 「**送信**」ボタンをクリック

::: info
お問い合わせを送信することで、プライバシーポリシーに同意したものとみなされます。
:::

送信後、運営者が内容を確認し、入力されたメールアドレス宛にご連絡いたします。
