# パートナーの登録

## ワークスペースとは

ディレクターがパートナーの稼働状況を管理・閲覧するための画面です。パートナーから受け取った共有トークンをワークスペースに登録することで、そのパートナーの稼働状況を閲覧できるようになります。

## パートナーの追加

![パートナー一覧](/images/workspace-list.png)

1. サイドバーの「**ワークスペース**」をクリック
2. 「**パートナー追加**」ボタンをクリック
3. パートナーから受け取ったトークンを入力
4. 必要に応じてグループを選択（未分類でも可）
5. 「**登録**」ボタンをクリック

![パートナー追加画面](/images/workspace-register.png)

::: info トークン登録時のルール
- 1つのトークンは1人のディレクターのみが登録できます
- 自分自身のトークンは登録できません
- 既に他のディレクターが登録済みのトークンは登録できません
:::

## パートナーの解除

1. ワークスペースのパートナー一覧で「**解除**」をクリック
2. 確認ダイアログで「OK」をクリック

解除されたトークンは「未登録」状態に戻り、別のディレクターが登録できるようになります。

## パートナーの並び替え

パートナー一覧では、ドラッグ&ドロップで並び順を変更できます。

- 各パートナーカード左側の **⠿** ハンドルをドラッグ
- グループ間の移動もドラッグ&ドロップで可能
- 変更は自動的に保存されます
