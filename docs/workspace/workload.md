# 稼働状況の閲覧

## 稼働状況一覧

ワークスペースの「**稼働状況一覧**」タブでは、登録済みの全パートナーの稼働状況をテーブル形式で一覧表示します。

- **グループヘッダー**: パートナーがグループ分けされている場合、グループ名が行として表示されます
- **パートナー行**: 各パートナーの日別合計稼働率が表示されます
- **色の濃淡**: 稼働率が高いほど色が濃くなります
- **赤色表示**: 稼働率が100%以上の日

月の切り替えは画面上部の「**← 前月**」「**次月 →**」で行えます。

![稼働状況一覧](/images/workspace-workload.png)

## 個別パートナーの稼働状況

パートナー名をクリックすると、そのパートナーの稼働状況を個別に閲覧できます。

![個別パートナーの稼働状況](/images/workspace-show.png)

::: info 表示される情報
共有ページでは**合計稼働率のみ**が表示されます。パートナーのタスク名、案件名、クライアント名などの詳細情報は一切表示されません。
:::

## 稼働率の読み方

| 稼働率 | 意味 |
|--------|------|
| 0% | 完全に空いている |
| 50% | 半日程度の空きがある |
| 100%以上 | 稼働枠が埋まっている（赤色で警告） |

::: tip
稼働率にはタスクの工数と非稼働日の両方が含まれます。例えば、タスク工数60% + 非稼働日40% = 表示上100%（稼働枠なし）となります。
:::
