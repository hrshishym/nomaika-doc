# 稼働状況の閲覧

## 稼働状況一覧

ワークスペースの「**稼働状況一覧**」タブでは、登録済みの全パートナーの稼働状況をテーブル形式で一覧表示します。

- **グループヘッダー**: パートナーがグループ分けされている場合、グループ名が行として表示されます
- **パートナー行**: 各パートナーの日別の稼働ステータスが色で表示されます
- **ステータス色**: 稼働率に応じて4段階のステータス色でセルが塗り分けられます

月の切り替えは画面上部の「**←**」「**→**」ボタン、または年・月のセレクトボックスから行えます。

![稼働状況一覧](/images/workspace-workload.png)

## 稼働ステータスの読み方

稼働率に応じて、以下の4つのステータスで表示されます。

| ステータス | 稼働率 | 意味 |
|--------|--------|------|
| 空いています | 0〜45% | 十分な空き枠があります |
| 作業可能 | 46〜75% | 作業をお引き受けできます |
| 要相談 | 76〜95% | 軽微な作業のみ対応可能です |
| 埋まっています | 96%以上 | 稼働枠が埋まっています |

::: tip
稼働率にはタスクの工数と非稼働日の両方が含まれます。例えば、タスク工数60% + 非稼働日40% = 表示上100%（埋まっています）となります。
:::

::: info プライバシーへの配慮
稼働状況一覧では具体的な稼働率（%）は表示されず、ステータスのみが共有されます。パートナーのタスク名、案件名、クライアント名などの詳細情報も一切表示されません。
:::

## 個別パートナーの稼働状況

パートナー名をクリックすると、そのパートナーの稼働状況を個別に閲覧できます。

![個別パートナーの稼働状況](/images/workspace-show.png)

個別ビューでは「**テーブル**」「**カレンダー**」の2つの表示形式を切り替えできます。
