# グループ管理

## グループとは

ワークスペースに登録したパートナーを「デザイナー」「コーダー」「ディレクター」などの任意のグループで分類する機能です。

- グループ名は自由に設定できます
- グループの表示順はドラッグ&ドロップで変更可能
- グループに属さないパートナーは「**未分類**」として最下部に表示されます

## グループの作成

1. ワークスペース画面の「**グループ管理**」をクリック
2. 「**新しいグループを作成**」セクションでグループ名を入力
3. 「**作成**」ボタンをクリック

::: info
同じ名前のグループは作成できません。
:::

## グループ名の変更

1. グループ管理画面でグループの「**編集**」をクリック
2. 新しいグループ名を入力
3. 「**保存**」をクリック

## グループの削除

1. グループ管理画面でグループの「**削除**」をクリック
2. 確認ダイアログで「OK」をクリック

::: warning
グループを削除すると、そのグループに所属していたパートナーは「**未分類**」に移動します。パートナーの登録自体は解除されません。
:::

## グループの並び替え

グループ管理画面で、各グループの **⠿** ハンドルをドラッグ&ドロップして表示順を変更できます。変更は自動的に保存されます。

## パートナーのグループ変更

パートナー一覧画面で、パートナーカードをドラッグして別のグループに移動できます。

1. ワークスペースの「**パートナー一覧**」タブを開く
2. 移動したいパートナーの **⠿** ハンドルをドラッグ
3. 移動先のグループにドロップ
